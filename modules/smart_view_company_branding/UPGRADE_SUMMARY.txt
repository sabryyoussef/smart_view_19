INSTALLATION INSTRUCTIONS
================================================================================

1. Ensure you're running Odoo 19.0
2. Copy the module to your addons directory:
   /home/sabry3/odoo_19_comunity/custom_addons/company_header_footer/

3. Update addons path in odoo.conf:
   addons_path = /path/to/odoo19/addons,/path/to/custom_addons

4. Restart Odoo server

5. Update Apps List:
   - Go to Apps menu
   - Click "Update Apps List"

6. Install Module:
   - Search for "Company Header Footer"
   - Click Install
